#  News Website - Next.js Assignment

A responsive **News Website** built using **Next.js**, which fetches and displays the latest news from [NewsAPI.org](https://newsapi.org/).  
This project includes category-based filtering, search functionality, pagination, and clean UI using TailwindCSS.

---

##  Features

✅ Display top headlines  
✅ Filter news by category (Business, Technology, Sports, etc.)  
✅ Search for articles by keyword  
✅ Pagination for more results  
✅ News detail view with full content  
✅ Responsive layout (mobile & desktop)  
✅ Error and loading state handling  

---

## Tech Stack

- **Framework:** Next.js  
- **Language:** JavaScript (React)  
- **Styling:** CSS3 
- **API Calls:** Axios  
- **State Management:** React Hooks (`useState`, `useEffect`)  
- **Testing:** Jest & React Testing Library
